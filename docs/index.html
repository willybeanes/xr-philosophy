What is xR? Expected Runs (xR) uses Statcast exit velocity and launch angle to estimate how many runs each team's batted balls deserved to produce, independent of where the ball landed. For walks, strikeouts, and other non-batted events, xR uses historical run values. A red row means the team with higher xR lost. Click any game to see the cumulative xR chart.
